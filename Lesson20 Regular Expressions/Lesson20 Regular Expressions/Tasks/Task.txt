Задание 1 Задачи на регулярные выражения
1.	Написать регулярное выражение, определяющее является ли данная строка строкой «abcdefghijklmnopqrstuv18340» или нет. 
– пример правильных выражений: abcdefghijklmnopqrstuv18340. 
– пример неправильных выражений: abcdefghijklmnoasdfasdpqrstuv18340.
2.	Написать регулярное выражение, определяющее является ли данная строка GUID с или без скобок. 
Где GUID это строчка, состоящая из 8, 4, 4, 4, 12 шестнадцатеричных цифр разделенных тире. 
– пример правильных выражений: e02fd0e4-00fd-090A-ca30-0d00a0038ba0. 
– пример неправильных выражений: e02fd0e400fd090Aca300d00a0038ba0.
3.	Написать регулярное выражение, определяющее является ли заданная строка правильным MAC-адресом. 
– пример правильных выражений:    aE:dC:cA:56:76:54
– пример неправильных выражений: 01:23:45:67:89:Az (буква z)
4.	Написать регулярное выражение, определяющее является ли данная строчка шестнадцатеричным идентификатором 
цвета в HTML. Где #FFFFFF для белого, #000000 для черного, #FF0000 для красного и т.д. 
– пример правильных выражений: #FFFFFF, #FF3421, #00ff00. 
– пример неправильных выражений: 232323, f#fddee, #fd2.
5.	Написать регулярное выражение, определяющее является ли данная строчка датой в формате dd/mm/yyyy. Начиная с 2000 года до 9999 года. 
– пример правильных выражений: 29/02/2000, 30/04/2003, 01/01/2003. 
– пример неправильных выражений: 30-04-2003, 1/1/1899.
6.	Проверить, надежно ли составлен пароль. Пароль считается надежным, если он состоит из 8 или более символов. 
Где символом может быть английская буква, цифра и знак подчеркивания. Пароль должен содержать хотя бы одну 
заглавную букву, одну маленькую букву и одну цифру. 
– пример правильных выражений: C00l_Pass, SupperPas1. 
– пример неправильных выражений: Cool_pass, C00l.
7.	Проверить является ли заданная строка шестизначным числом, записанным в десятичной системе 
счисления без нулей в старших разрядах. 
– пример правильных выражений: 123456, 234567. 
– пример неправильных выражений: 1234567, 12345.
8.	Есть текст со списками цен. Извлечь из него цены в USD, RUR, EU. 
– пример правильных выражений: 23.78 USD. 
– пример неправильных выражений: 22 UDD, 0.002 USD.
9.	Проверить существуют ли в тексте цифры, за которыми не стоит «+». 
– пример правильных выражений: (3 + 5) – 9 × 4. 
– пример неправильных выражений: 2 * 9 – 6 × 5.
10.	Создать запрос для вывода только правильно написанных выражений со скобками 
(количество открытых и закрытых скобок должно быть одинаково). 
– пример правильных выражений: (3 + 5) – 9 × 4. 
– пример неправильных выражений: ((3 + 5) – 9 × 4.


Задание 2 Программа «Погода».
C использованием регулярных выражений реализовать правильный вывод значений температур и 
других параметров, заменить слова, написанные на языке отличном от русского языка. 
Получение погоды реализовать с применением кода из приложения:

// Пример работы с GisMeteo 
// сслыка на получение данных 26850 - код Минска
string path = @"http://informer.gismeteo.by/rss/26850.xml";
XDocument doc = XDocument.Load(path);
// Вывод информации о сайте
Console.WriteLine(doc.Root.Element("channel").Element("description").Value);

// получение информации о дочерних элементах  элемента channel
foreach (var item in doc.Root.Element("channel").Elements())
{
    if (item.Name == "item")// проверка на элемент item
    {
      Console.WriteLine(item.Element("title").Value);  // вывод информации о городе, дате
      Console.WriteLine(item.Element("description").Value);// вывод ифнормации о погоде
    }
}
